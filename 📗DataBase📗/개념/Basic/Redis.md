## 💡 Redis

- Key-Value Store NoSql DB이다
- Single Thread로 동작하며 자료구조를 지원함
- 다양한 용도로 사용될 수 있도록 다양한 기능 지원
- 데이터의 스냅샷 & AOF 로그를 통해 복구가 가능해서 어느정도 영속성도 보장됨
- 스프링에선 세션을 관리하거나, 캐싱을 하는데에 자주 사용된다



### Redis와 Memcached의 차이점

- Redis
  - 싱글 스레드 동작
  - 다양한 자료구조 지원
  - 스냅샷, AOF 로그 등 다양한 기능 지원
- Memcached
  - 멀티 스레드 지원, 멀티 프로세싱 가능
  - 문자열 형태로만 저장 가능
