## 유저 생성

```sql
create database DB명;
use DB명;

create database dains character set utf8mb4 collate utf8mb4_general_ci;

create user '아이디'@'%' identified by '비밀번호';
grant all privileges on DB명.* to '아이디'@'%';
flush privileges;
```

---

## 유저 삭제

```shell
drop user 사용자@서버;

EX:
drop user test@localhost;
```
