## Vue 3

1. **성능 개선**: Vue 3는 Vue 2에 비해 초기 렌더링 속도가 2배 빠르고, 업데이트 성능도 3배 빠르게 향상되었습니다.
    
2. **Composition API**: 이는 Vue의 가장 주요한 새로운 기능 중 하나로, 코드 재사용과 로직의 조직을 개선하기 위한 API입니다. `ref`, `reactive`, `watch`, `computed` 등의 함수가 포함됩니다.
    
3. **Fragment**: Vue 3에서는 단일 컴포넌트에서 여러 root 요소를 가질 수 있습니다.
    
4. **Teleport (또는 Portal)**: Vue에서 특정 컴포넌트의 일부를 다른 DOM 위치로 "전송"할 수 있게 해줍니다.
    
5. **Suspense**: 데이터를 기다리는 동안 다른 컴포넌트를 보여주게 해주는 기능으로, 비동기 컴포넌트 로딩을 용이하게 합니다.
    
6. **TypeScript 지원**: Vue 3는 TypeScript로 작성되었으며, TypeScript와의 통합이 향상되었습니다.
    
7. **Custom Renderer API**: 사용자 정의 렌더러를 작성하는 것을 더 쉽게 만들어 줍니다.
    
8. **Proxy 기반 반응형 시스템**: Vue 3는 ES6의 Proxy를 사용하여 반응형 시스템을 다시 작성했기 때문에 성능 향상과 잘못된 사용을 방지하는 데 도움이 됩니다.
    
9. **V-model 개선**: Vue 3에서는 동일한 컴포넌트 내에서 여러 `v-model`을 사용할 수 있게 되었습니다.
    
10. **API 변경 및 개선**: 여러 내부 API 및 lifecycle 메서드가 조정되거나 변경되었습니다.
    
11. **Size 최적화**: Vue 3는 트리 쉐이킹에 더 적합하도록 설계되었으므로, 필요한 기능만 포함하여 빌드 사이즈를 줄일 수 있습니다.
    
12. **Improved Slot API**: 슬롯의 사용 및 구문이 개선되어 더 간결하고 예측 가능하게 되었습니다.
    
13. **Improved Custom Directive API**: 사용자 정의 지시문 API가 변경되어 더 일관성 있고 예측 가능하게 되었습니다.
    
14. **Migration Build**: Vue 2에서 Vue 3로 이동하면서 발생할 수 있는 문제점을 찾아서 알려주는 도구입니다.
