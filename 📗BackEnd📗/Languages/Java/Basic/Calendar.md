## **Calendar.add()**

- 첫번쨰 파라미터 : Calendar의 클래스 상수
- 두번쨰 파라미터 : 더할 값

<br>

1. **`Calendar.YEAR` (1):**
    - 설명: 연도를 나타내는 필드입니다.
2. **`Calendar.MONTH` (2):**
    - 설명: 월을 나타내는 필드로, 0부터 시작합니다 (0은 1월, 1은 2월, ..., 11은 12월).
3. **`Calendar.DAY_OF_MONTH` (5):**
    - 설명: 월의 날짜를 나타내는 필드로, 1부터 시작합니다.
4. **`Calendar.HOUR_OF_DAY` (11):**
    - 설명: 24시간 형식으로 표시된 시간의 시간 부분을 나타내는 필드입니다.
5. **`Calendar.MINUTE` (12):**
    - 설명: 시간의 분 부분을 나타내는 필드입니다.
6. **`Calendar.SECOND` (13):**
    - 설명: 시간의 초 부분을 나타내는 필드입니다.
7. **`Calendar.MILLISECOND` (14):**
    - 설명: 시간의 밀리초 부분을 나타내는 필드입니다.
