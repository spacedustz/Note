## 📘 Throughput Optimization

이번에는 성능 정의 중 2번째인 처리량에 대해 알아보겠습니다.

처리량이란 **주어진 기간에 완성되는 작업의 수**를 뜻합니다, 보통 처리량은 시간 단위 / 초 단위로 작업을 나눠 측정합니다.

<br>

하나의 작업을 완수하는데 T의 시간이 걸리면 이룰수 있는 최소 처리량은 1/T 가 됩니다.

작업은 하위 Task로 나눌 수 있다면 T/N

- 처리량 향상을 위한 2가지 접근법
- Thread Pooling
