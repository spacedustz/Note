## Spring AOP

> **AopConfig**

AO

```java
@Configuration  
@EnableAspectJAutoProxy  
public class AopConfig {  
    @Bean  
    public EventAspect aspect() {  
        return new EventAspect();  
    }  
}
```

> **EventAspect**

```java
@Slf4j  
@Aspect  
public class EventAspect {  
    @Pointcut("execution(public void com.accesscontrol.service.EventService.*(..))")  
    public void targetEvent() {}  
  
    @Around(value = "targetEvent()")  
    public Object eventAspect(ProceedingJoinPoint joinPoint) throws Throwable {  
        long start = System.currentTimeMillis();  
        log.info("Aspect 실행 시작 - {}", start);  
  
        Object result = null;  
        try {  
            result = joinPoint.proceed();  
        } catch (Exception e) {  
            log.error("Event Aspect Error - {}", e.getMessage());  
        } finally {  
            long end = System.currentTimeMillis();  
            Signature signature = joinPoint.getSignature();  
            log.info("Aspect Method 호출 : {}, {}", joinPoint.getTarget().getClass().getSimpleName(), signature.getName());  
            log.info("Aspect 실행 끝 - {}", end);  
        }  
  
        log.info("Event Aspect 실행 - {}", result);  
        return result;  
    }  
}
```
