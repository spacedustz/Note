## HTML 파일 브라우저에서 열기

## 방법 1. 폴더에서 열기

가장 기본적인 방법입니다. HTML 파일이 있는 폴더로 이동한 후 실행하면 기본 브라우저로 실행됩니다.

![image](https://s3.ap-northeast-2.amazonaws.com/urclass-images/YSNEAUo6xZDY9WOPTXWmp-1675935569667.png)

## 방법 2. VScode Extension 활용하기 (Live Server)

VScode에는 개발자가 사용할 수 있는 다양한 Extension을 설치할 수 있습니다. 그중 Live Server라는 유용한 Extension을 설치하고 사용하는 방법을 알아보겠습니다.

1.  VScode 왼쪽 바에서 Extension 메뉴를 선택하고 상단에 있는 검색창에 Live Server를 검색하여 설치합니다.

![image](https://s3.ap-northeast-2.amazonaws.com/urclass-images/M2Pkod5uX9d3k47JidKdI-1675935532263.gif)

2.  설치가 되었다면, VScode에서 파일을 오른쪽 클릭한 후 Open with Live Server를 클릭하면 브라우저에서 HTML 파일을 열 수 있습니다.
    

![image](https://s3.ap-northeast-2.amazonaws.com/urclass-images/ftntdCwCzARpD_5-PPpar-1675935717254.gif)

3. Live Server는 HTML 파일 수정 후 브라우저를 새로 고치지 않아도 변경 사항이 바로 적용되는 장점이 있습니다.

![image](https://s3.ap-northeast-2.amazonaws.com/urclass-images/aVVkuio9T_r_E9hgFV_vk-1675935493943.gif)
