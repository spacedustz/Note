## 💡 NCP -> AWS Server Migration

Ncloud 크레딧 만료 기한 임박으로 인한 서버 이전 계획 작성 (2023.06.30 예정) -> AWS로 서버 이전 예정

<br>

**기존 서버 스펙** 

- (CPU 2 RAM 8GB) - Spring
- (CPU 2 RAM 8GB) - MySQL

<br>

**요구사항**

- Docker 사용 예정 / 추후 Travis 등 빌드 배포 자동화 툴 고려
- HTTPS 사용 예정 (기본 EC2 - 배포)
- 로드 밸런서 사용 예정
- 같은 성능으로 구현하되 확장성 고려
