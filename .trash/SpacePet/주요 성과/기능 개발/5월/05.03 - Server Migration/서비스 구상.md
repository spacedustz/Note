## 서비스 구상

### EC2 (Dedicated Host  or  Reserved Instance)

- 개발서버, 배포서버, 웹 어드민서버, Nginx 서버
- 4개 인스턴스
- Elastic IP (무료)
- Reserved Instance를 기준으로 가격 측정
- Dedicated Host도 고려

<br>

### S3 Bucket

- Standard
  - 검색이 주 용도, 데이터 단기 보관에 적합
- Glacier 
  - 검색이 주 용도, 데이터 장기 보관에 적합
- Standard_IA 
  - 주 용도 = 자주 액세스 하지 않지만 즉시 액세스 필요한 데이터 일 경우, 3곳의 AZ에 분산 저장됨
- OneZone_IA 
  - 주 용도 = Standard와 동일하지만 1곳의 AZ 저장으로 인한 정보 손실 가능성
- Intelligent Tiering 
  - 불규칙한 액세스 빈도를 가진 서비스일 경우 적합
- Glacier Deep Archive 
  - 데이터를 10년 이상 장기 보관 시 적합
