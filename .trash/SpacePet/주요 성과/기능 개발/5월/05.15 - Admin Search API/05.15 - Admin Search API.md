## 💡 Admin Search API

---

## 요구사항

-   같은 API로 유저 검색
    
-   파라미터 이름당 정회원 / 부회원 / 준회원을검색 (eg ?member=준회원)
    
-   항상 접속순으로 결과를 반환
    
-   삭제된 유저도 검색
    
-   15명씩 검색하게 구성한다 (페이징)
    
-   어드민만 접근 가능
    
-   기존 admin/users 컨트롤러와 같은 역할을 하되 위 조건을 준수

<br>

-   같은 기능을 하되 여러 검색 파라미터를 추가
    
-   모든 조건은 AND 조건이며 null 혹은 ““ (빈칸) 조건으로 들어올 시 그 조건은 무시 (반영 X)
    
-   유저 이름 / 동물 이름 / 동물 타입 검색
    
-   페이징은 유지

<br>

-   지역관련 검색기능 추가
    
-   서울특별시 입력시 → 서울특별시 유저 전체 반환
    
-   파라미터 (서울특별시 서초구)로 입력시 서울특별시 서초구 반환
    
-   봉천동 입력시 - 시와 구에 관계없이 봉천동 유저 반환
    
-   서초구 봉천동 입력시 → 없는 유저이므로 반환 0로 반환 (200코드로 반환,throw X)
    
-   `List<AdUserDTO> 반환`

## 경우의 수 생각해보기

서울특별시, 인천광역시 = specialSido로 따로 분류하고 다른 시 들과 구분

1. xx 시, xx 도
2. xx구, xx시 xx구
3. xx동


최대 Tokenizer 4

Token이 1인 경우의 수
- xx도 - 완료
- xx시 - 완료
- xx구 - 완료
- xx군 - 완료
- xx동 - 완료


Token이 2개인 경우의 수
- xx시 xx구,군 - 완료
- xx구,군 xx동 - 완료
- (xx시 xx구) 만 들어왔을때 - 완료
- xx도 xx시 - 완료


Token이 3개인 경우의 수
- xx도 (xx시 xx구) - 완료 (일반시 기준)
- (xx시 xx구) xx동 - 완료 (일반시 기준)


Token이 4개인 경우의 수
-  xx도 (xx시 xx구) xx동

### CustomUserRepositoryImpl

```kotlin
/**  
 * @author 신건우  
 * @param sido : 시도 정보  
 * @param sigungu : 시군구 정보  
 * @param dong : 동 정보  
 * @param grade : 회원 등급  
 * @param userName : 회원 아이디  
 * @param animalName : 반려동물 이름  
 * @param animalType : 반려동물 타입  
 * @param paging : 페이지 정보  
 * @desc 회원 등급 기준 15명씩 검색 (어드민 기능, 삭제된 유저 포함)  
 */
 override fun findByAllUserForAdminOrderByTokenCreatedAt(  
    sido: String?,  
    sigungu: String?,  
    dong: String?,  
    grade: String?,  
    userName: String?,  
    animalName: String?,  
    animalType: String?,  
    paging: Pageable  
): List<User> {  
    val userBuilder = BooleanBuilder()  
    val animalBuilder = BooleanBuilder()  
  
    if (!sido.isNullOrEmpty()) {  
        userBuilder.or(user.geometry.sido.eq(sido))  
        animalBuilder.or(animal1.user.geometry.sido.eq(sido))  
    }  
  
    if (!sigungu.isNullOrEmpty()) {  
        userBuilder.or(user.geometry.sigungu.contains(sigungu))  
        animalBuilder.or(animal1.user.geometry.sigungu.contains(sigungu))  
    }  
  
    if (!dong.isNullOrEmpty()) {  
        userBuilder.or(user.geometry.dong.eq(dong))  
        animalBuilder.or(animal1.user.geometry.dong.eq(dong))  
    }  
  
    if (!grade.isNullOrEmpty()) {  
        userBuilder.or(user.grade.eq(grade))  
        animalBuilder.or(animal1.user.grade.eq(grade))  
    }  
  
    if (!userName.isNullOrEmpty()) {  
        userBuilder.and(user.name.eq(userName))  
        animalBuilder.and(animal1.user.name.eq(userName))  
    }  
      
    if (!animalName.isNullOrEmpty()) {  
        animalBuilder.and(animal1.name.like("%$animalName%", '!'))  
    }  
  
    if (!animalType.isNullOrEmpty()) {  
        animalBuilder.and(animal1.animal.eq(animalType))  
    }  
  
    val userQuery = from(user)  
        .where(userBuilder)  
        .orderBy(user.userToken.tokenCreatedAt.desc())  
        .offset(paging.offset)  
        .limit(paging.pageSize.toLong())  
  
    val animalQuery = from(animal1)  
        .where(animalBuilder)  
        .offset(paging.offset)  
        .limit(paging.pageSize.toLong())  
        .orderBy(animal1.user.userToken.tokenCreatedAt.desc())  
  
    val users = userQuery.fetch()  
  
    if (!animalName.isNullOrEmpty() || !animalType.isNullOrEmpty()) {  
        val animals = animalQuery.fetch()  
        return animals.stream()  
            .filter { it.user != null }  
            .map { it.user }  
            .collect(Collectors.toList()) as List<User>  
    }  
  
    return users  
}
```

### AdminService

```kotlin
/**  
 * @author 신건우  
 * @param location : 위치 정보  
 * @param grade : 회원 등급  
 * @param userName : 회원 아이디  
 * @param animalName : 반려동물 이름  
 * @param animalType : 반려동물 타입  
 * @param index : 페이지 개수  
 * @param page : 글 개수  
 * @desc 회원 등급 기준 15명씩 검색 (어드민 기능, 삭제된 유저 포함)  
 */
 fun findAllUserForAdminOrderByTokenCreatedAt(  
    location: String?,  
    grade: String?,  
    userName: String?,  
    animalName: String?,  
    animalType: String?,  
    index: Int,  
    page: Int  
): List<AdUserDTO> {  
  
  
    val paging = PageRequest.of(index, page)  
  
    val parts: StringTokenizer? = if (location != null ) StringTokenizer(location, " ") else null  
    var sido: String? = ""  
    var sigungu: String? = ""  
    var dong: String? = ""  
  
    var specialSido = listOf("서울특별시", "인천광역시")  
  
    while (parts?.hasMoreTokens() == true) {  
  
        if (parts.countTokens() == 1) {  
            sido = parts.nextToken()  
  
            // xx구 or xx군으로 들어왔을때  
            if (sido.endsWith("구") || sido.endsWith("군")) {  
                sigungu = sido  
                sido = ""  
            }  
            // xx동만 들어왔을때  
            else if (sido.endsWith("동")) {  
                dong = sido  
                sido = ""  
            }  
            // e.g 성남시 (시군구인 시가 들어왔을때)  
            else if (!specialSido.contains(sido)) {  
                sigungu = sido  
                sido = ""  
            }  
        }  
  
        if (parts.countTokens() == 2) {  
            sido = parts.nextToken()  
            sigungu = parts.nextToken()  
  
            // xx구,군 xx동만 들어왔을때  
            if ((sido.endsWith("구") || sido.endsWith("군")) && sigungu.endsWith("동")) {  
                dong = sigungu  
                sigungu = sido  
                sido = ""  
            }  
            // (xx시 xx구)만 들어왔을때 (e.g. 성남시 분당구)  
            else if ((sido.endsWith("시") && !specialSido.contains(sido)) && sigungu.endsWith("구")) {  
                sigungu = "$sido $sigungu"  
                sido = ""  
            }  
            // xx시 xx구,군만 들어왔을때 (일반시 기준)  
            else if (!specialSido.contains(sido) && (sigungu.endsWith("구") || sigungu.endsWith("군"))) {  
                sigungu = "$sido $sigungu"  
                sido = ""  
            }  
            // xx도 xx시  
            else if (sido.endsWith("도") && (sigungu.endsWith("구") && sigungu.endsWith("군"))) Unit  
        }  
  
        if (parts.countTokens() == 3) {  
            sido = parts.nextToken()  
            sigungu = parts.nextToken()  
            dong = parts.nextToken()  
  
            // (일반시 기준) xx도 xx시,군  
            if (!specialSido.contains(sigungu) && (dong.endsWith("구") || dong.endsWith("군"))) {  
                sigungu = "$sigungu $dong"  
                dong = ""  
            }  
            // (일반시 기준) (xx시 xx구,군) xx동  
            else if (!specialSido.contains(sido) && (sigungu.endsWith("구") || sigungu.endsWith("군"))) {  
                sigungu = "$sido $sigungu"  
                sido = ""  
            }  
        }  
  
        // xx도 xx시 xx구 xx동 이 한번에 들어왔을때  
        if (parts.countTokens() == 4) {  
            sido = parts.nextToken()  
            sigungu = parts.nextToken() + " " + parts.nextToken()  
            dong = parts.nextToken()  
        }  
    }  
  
    val result = try {  
        userRepository.findByAllUserForAdminOrderByTokenCreatedAt(sido, sigungu, dong, grade, userName, animalName, animalType, paging)  
    } catch (e: Exception) {  
        log.error("[ Admin ] - 회원 리스트 검색 실패 - $e, ${e.printStackTrace()}")  
        throw CommonException("ADMIN-003", HttpStatus.BAD_REQUEST)  
    }  
  
    log.info("[ Admin ] - 회원 리스트 검색 성공, 검색 기반 위치 : [ $sido $sigungu $dong ], 결과값 수 : [ ${result.size} ]")  
  
    return result.stream().map { AdUserDTO.ModelMapper.entityToDto(it) }.toList()  
}
```

```kotlin
/**  
 * @author 신건우  
 * @param location : 위치 정보  
 * @param grade : 회원 등급  
 * @param userName : 회원 아이디  
 * @param animalName : 반려동물 이름  
 * @param animalType : 반려동물 타입

 */
 @Auth(role = ADMIN)  
@GetMapping("/Neousers")
fun findUsersByGeometry(  
    @RequestParam("location") location: String?,  
    @RequestParam("grade") grade: String?,  
    @RequestParam("userName") userName: String?,  
    @RequestParam("animalName") animalName: String?,  
    @RequestParam("animalType") animalType: String?,  
    @RequestParam("index", defaultValue = "0") index: Int,  
    @RequestParam("page", defaultValue = "15") page: Int  
): ResponseEntity<List<AdUserDTO>> {  
    return ResponseEntity(adminService.findAllUserForAdminOrderByTokenCreatedAt(location, grade, userName, animalName, animalType, index, page), HttpStatus.OK)  
}
```
