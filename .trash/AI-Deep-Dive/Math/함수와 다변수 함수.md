f = 함수

y = 결과값

### 1개의 입출력

`x -> f -> f(x) = y`

x = 입력
f = 함수
f(x) = 출력

<br>

### 2개의 입력 1개의 출력

`x,y -> f -> f(x,y) = z = yx^2`

<br>

### 1개의 입력 2개의 출력

`x -> f -> f(f, f^2) = x, x^2 = f_(x) `
