## 📘 개인 공부 메모 Repository

<br>

> 📕 **기능개발**

- Event Simulator : Message Broker인 RabbitMQ를 이용한 부하테스트 용도의 시뮬레이터 개발
- 출입 통제 시스템 : Event 기반 어플리케이션에서 Message Queue를 이용한 이벤트 데이터 처리를 통한 출입 통제 기능 개발

<br>

> 📕 **Algorithm**

- 알고리즘의 개념과 풀이를 기록한 카테고리입니다.

<br>

> 📕 **BackEnd**
- Converter : 범용 포맷의 데이터 파싱/변환, FFmpeg을 이용한 RTSP to HLS 영상 스트리밍
- Error Handling : 에러를 해결한 기록
- Frameworks : 프레임워크에서 사용한 기술들
- Languagues : 백엔드에서 사용한 언어들의 개념과 활용 등 (Java, Kotlin, Python)
- Message Broker : RabbitMQ를 사용하며 학습한 내용들

<br>

> 📕 **Computer Science**

- 기본적인 OS, Hardware, Software의 CS 지식들을 모아놓은 공간

<br>

> 📕 **Database**

- SQL, NoSQL 기반 다양한 데이터베이스 학습

<br>

> 📕 FrontEnd

- Frameworks : 프론트엔드 프레임워크 학습
- Languages : 프론트엔드 개발을 위한 언어의 개념, 활용 학습 (JavaScript, TypeScript, Dart)

<br>

> 📕 **Ops-Tools**

- AWS : 서버리스 어플리케이션 등 클라우드 기반 서버 아키텍처의 사용을 위해 학습
- CI & CD : Github Actions, Jenkins (Blue/Green Deploy) 등 자동화된 배포를 위해 학습
- Docker : Docker, Docker Compose, Docker Swarm 복습
- Management & Tool : 개발을 할떄 사용한 툴들의 사용법
- MSA : Kubernetes 복습
